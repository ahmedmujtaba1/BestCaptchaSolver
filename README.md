# Chrome Extension

Download [ZIP](https://github.com/noCaptchaAi/noCaptcha_extension/archive/refs/heads/main.zip)

<p>
Extract and then:

- 1: Enable Chrome Developer mode > chrome://extensions
- 2: Load Unpacked
- 3: Click on api setup button from https://dash.nocaptchaai.com
